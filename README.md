# System_of_nonlinear_equations

Данный проект реализует решение системы нелинейных алгебраических уравнений четырьмя итерационными методами с визуализацией для выбора начального приближения и сравнительным анализом качества и скорости сходимости.

В программе реализуются следующие шаги:

1. Визуализация

- Графики функций — для визуального определения областей существования решений
- Линии уровня суммы квадратов невязок — для выбора начальных приближений и анализа поведения итерационного процесса

2. Реализация четырех методов решения

- Простая итерация: последовательное приближение по явной схеме 
- Метод Ньютона: линеаризация с обновлением матрицы Якоби на каждой итерации 
- Модифицированный метод Ньютона: постоянная матрица производных (вычисляется один раз) 
- Дискретный метод Ньютона: разностная аппроксимация производных  

3. Сравнительный анализ

- Оценка количества итераций до достижения заданной точности
- Анализ нормы невязки
